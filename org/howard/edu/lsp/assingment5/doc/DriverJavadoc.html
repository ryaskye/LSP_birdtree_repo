Class Overview

This class demonstrates the use of the IntegerSet class by performing several operations such as union, intersection, difference, and others. The main method showcases the use of the IntegerSet class and its methods.

Method: main

The main method is the entry point of the Driver class. It demonstrates the creation and manipulation of IntegerSet objects. It performs the union, intersection, and difference operations and tests various methods like add(), remove(), contains(), and isEmpty().

Method Signature:

    public static void main(String[] args)

Description:

The main method performs the following operations:

  • Performs union between Set1 and Set2.

  • Performs the difference between Set1 and Set2.

Code Walkthrough

The main method is divided into several parts, each of which tests a different functionality of the IntegerSet class:

  • Set1 creation and testing: The method creates set1, adds elements 1, 2, and 3 to it, and then prints the values in set1 along with its smallest and largest elements.
  • Set2 creation and testing: The method creates set2, adds elements 3, 4, and 5 to it, and then performs the union of set1 and set2, printing the results.
  • Intersection operation: Clears set1, adds 1, 2, and 3, and then performs the intersection with set2, printing the resulting set.
  • Difference operation: Clears set1, adds 1, 2, and 3, and then performs the difference operation between set1 and set2, printing the result.
  • Testing contains, remove, and isEmpty: Tests the contains(), remove(), and isEmpty() methods, printing the results after adding and removing elements from set1.

Code Example:

    public static void main(String[] args) {
        IntegerSet set1 = new IntegerSet();
        set1.add(1);
        set1.add(2);
        set1.add(3);
        
        System.out.println("Value of Set1 is: " + set1.toString());
        System.out.println("Smallest value in Set1 is: " + set1.smallest());
        System.out.println("Largest value in Set1 is: " + set1.largest());
       
        IntegerSet set2 = new IntegerSet();
        set2.add(3);
        set2.add(4);
        set2.add(5);
        
        System.out.println("\nUnion of Set1 and Set2");
        set1.union(set2);
        System.out.println("Result of union of Set1 and Set2: " + set1.toString());

        set1.clear();
        set1.add(1);
        set1.add(2);
        set1.add(3);

        System.out.println("\nIntersection of Set1 and Set2");
        set1.intersect(set2);
        System.out.println("Result of intersection of Set1 and Set2: " + set1.toString());

        set1.clear();
        set1.add(1);
        set1.add(2);
        set1.add(3);

        System.out.println("\nDifference of Set1 - Set2");
        set1.diff(set2);
        System.out.println("Result of difference of Set1 - Set2: " + set1.toString());

        System.out.println("\nTesting contains, remove, and isEmpty");
        set1.add(2);
        System.out.println("Set1 contains 2: " + set1.contains(2));
        set1.remove(2);
        System.out.println("Set1 contains 2 after removal: " + set1.contains(2));
        System.out.println("Is Set1 empty? " + set1.isEmpty());
        set1.clear();
        System.out.println("Is Set1 empty after clear? " + set1.isEmpty());
    }
